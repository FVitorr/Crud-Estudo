Este relatório faz parte de um projeto CRUD desenvolvido pela Célula Web, utilizando Angular para a construção da interface do usuário e Spring Boot para a criação da API de backend. O projeto visa fornecer uma solução eficiente e escalável para a gestão de dados de pessoas, permitindo a criação, leitura, atualização e exclusão de registros de forma intuitiva e ágil.

A Célula Web é uma empresa comprometida com a inovação e a excelência em desenvolvimento de software, oferecendo soluções personalizadas que atendem às necessidades específicas de cada cliente.
